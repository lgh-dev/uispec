# UISpec Do - 按规范开发

使用当前选定的 UISpec 设计规范开发功能。

## 使用方法

请描述你要开发的功能，我会严格按照选定的设计规范生成代码。

如果尚未选择规范，请先运行 /uispec-switch 选择一个设计规范。

## 设计规范要求

生成的代码将遵循以下要求：
- 色彩系统符合规范
- 间距系统符合规范
- 组件样式符合规范
- 交互效果符合规范
- 响应式设计
- 可访问性标准

## 执行流程（必须严格按顺序执行）

### 第一步：读取当前规范配置
从 `.uispec/current-spec.json` 读取当前选定的规范名称。

### 第二步：读取完整规范文件（关键！）
**CRITICAL**: 必须使用 Read 工具读取完整的规范文件，不能使用 limit 参数！

正确示例：
```javascript
Read('.uispec/specs/dark-elegant.md')  // ✅ 正确！读取全文
```

**规范文件包含以下完整内容：**
- 🎨 色彩系统（背景、文字、边框各5级层次）
- ✏️ 字体排版（Inter + JetBrains Mono）
- 📐 间距系统（4px 基数）
- 🔲 圆角系统
- 🎭 阴影系统（基础阴影 + 发光效果）
- 🎬 动画系统（3种时长、3种缓动曲线）
- 🧩 组件规范（按钮、输入框、卡片等 40+ 组件）

**必须读取完整规范后才能开始开发！**

### 第三步：开发功能
根据完整规范开发用户请求的功能。
