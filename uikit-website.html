UIKit
AI 时代的 UI 规范管理系统

一键安装，三个命令，五套规范
让 AI 生成的界面始终保持一致的设计语言

快速开始 查看源码 
# 一键安装
./quick-install.sh

# 选择设计规范
/uikit-switch

# 按规范开发
/uikit-do 创建登录页面

# 审查合规性
/uikit-check

强大而优雅

专为 AI 编程助手设计的 UI 规范管理系统

极速安装

UIKit CLI 采用 Bun 构建的原生二进制文件，一键安装。支持 Claude Code 和 Qoder 平台。

可扩展规范

内置五套专业设计规范，支持动态加载。你可以添加自己的规范文件，系统会自动识别。

多平台支持

目前支持 Claude Code 和 Qoder，未来将支持 Cursor、Windsurf 等更多 AI 编程工具。

实时生效

选择规范后立即生效，所有后续开发都会自动应用选定的设计系统。

智能审查

自动审查代码是否符合规范，提供详细的改进建议和修复方案。

框架无关

纯设计语言描述，支持 React、Vue、HTML 等任何技术栈，真正的通用解决方案。

三步上手

简单到不可思议

1

安装 UIKit CLI

一键安装命令行工具

./quick-install.sh
2

初始化到 AI 编程工具

选择你的 AI 编程平台

uikit init claude # 或 uikit init qoder
3

开始使用

重启 AI 工具后即可使用三个核心命令

/uikit-switch # 选择设计规范
