# UISpec Switch - 快速切换设计规范

## 可用规范

**dark-elegant** - 暗黑优雅 (开发IDE、专业软件)

## 使用方式

### 快速切换（推荐）
直接输入命令和规范名：
```
/uispec-switch dark-elegant
```

## 实现逻辑

将规范名保存到 `.uispec/current-spec.json`：
```json
{
  "spec": "dark-elegant"
}
```
